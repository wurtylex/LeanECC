Definition 1Code

A code \(C\) of block length \(n\) over an alphabet \(\alpha \) is a subset of \(\alpha ^{n}\). Typically, we will use \(q\) to denote the alphabet size \(|\Sigma |\).

Definition 3Dimension

The dimension \(k\) of a code \(C\) is

\[ k := \log _{q}(\lvert C\rvert ), \]

where \(q\) is the alphabet size and \(\lvert C\rvert \) is the number of codewords.

Definition 7Linear Code by Subspace

Let \(q\) be a prime power (i.e. \(q = p^{s}\) for some prime \(p\) and integer \(s \geq 1\)). \(C \subseteq \mathbb {F}^{n}_{q}\) is a linear code if it is a linear subspace of \(\mathbb {F}^{n}_{q}\). If \(C\) has dimenension \(k\) and distance \(d\) then it will be referred to as an \([n,k,d]_{q}\) or just an \([n,k]_{q}\) code. 1

Definition 2Alphabet Size

The alphabet size is \(q := \lvert \alpha \rvert \), the cardinality of the finite alphabet \(\alpha \).

LaTeX Lean
Definition 4Rate

The rate \(R\) of a code \(C\) is

\[ R := \frac{k}{n}. \]

where \(k\) is the dimension and block length of \(n\).

LaTeX Lean
Lemma 5Dimension is at most the block length

For every code \(C\) of block length \(n\), we have \(k \leq n\).

LaTeX Lean
Lemma 6Rate is at most one

For every code \(C\), we have \(R \leq 1\).
