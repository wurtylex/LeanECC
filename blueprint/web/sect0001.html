1 Basics of Codes

1.1 Definitions

Definition 1 Code
✓
#

Lean declarations

  • Code

A code \(C\) of block length \(n\) over an alphabet \(\alpha \) is a subset of \(\alpha ^{n}\). Typically, we will use \(q\) to denote the alphabet size \(|\Sigma |\).

Definition 2 Alphabet Size
✓
#

Lean declarations

  • Code.q

The alphabet size is \(q := \lvert \alpha \rvert \), the cardinality of the finite alphabet \(\alpha \).

Definition 3 Dimension
✓
#

Uses

  • Definition 1
  • Definition 2

Lean declarations

  • Code.dim

The dimension \(k\) of a code \(C\) is

\[ k := \log _{q}(\lvert C\rvert ), \]

where \(q\) is the alphabet size and \(\lvert C\rvert \) is the number of codewords.

Definition 4 Rate
✓
#

Uses

  • Definition 1
  • Definition 3

Lean declarations

  • Code.rate

The rate \(R\) of a code \(C\) is

\[ R := \frac{k}{n}. \]

where \(k\) is the dimension and block length of \(n\).

1.2 Properties

Lemma 5 Dimension is at most the block length
✓
#

Uses

  • Definition 3
  • Definition 2

Lean declarations

  • Code.dim_le_n

For every code \(C\) of block length \(n\), we have \(k \leq n\).

Proof ▶

Note that \(C \subseteq \alpha ^{n}\), we have \(\lvert C\rvert \le \lvert \alpha \rvert ^{n} = q^{n}\), thus \(k = \log _{q}\lvert C\rvert \le \log _{q} q^{n} = n\).

Lemma 6 Rate is at most one
✓
#

Uses

  • Definition 4
  • Lemma 5

Lean declarations

  • Code.rate_le_one

For every code \(C\), we have \(R \leq 1\).

Proof ▶

By Lemma 5, \(k \leq n\), and \(n \ge 0\). Therefore \(R = k/n \leq 1\).
